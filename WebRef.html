Computers

Basic Concepts

Byte: 8 bits
Bit: Basic unit of computerized information. It's a single binary (base 2) digit and can have a value of either 0 or 1.
Hexadecimal: Base 16. Each digit can have a value between 0 and 15 inclusive. Values 0 through 9 use the usual decimal representations and values 10 through 15 use capital letters A through F.
Hex: See Hexadecimal.
